### 👻 Ghost Chat — Encrypted Messaging on ESP32 + LoRa

</div>

> **An offline, encrypted peer-to-peer messaging system — no internet, no servers, just hardware.**

| Detail | Info |
|---|---|
| 🔧 **Hardware** | ESP32 Microcontroller + LoRa Radio Module |
| 🔐 **Security** | End-to-end encrypted messaging layer |
| 📡 **Range** | Long-range communication via LoRa protocol |
| 🌐 **Connectivity** | Zero internet dependency — fully offline |
| 💡 **Use Case** | Disaster zones, remote areas, privacy-first comms |

**What makes it special:**
- 🔒 Engineered a lightweight encryption layer ensuring message confidentiality without any internet infrastructure
- 📡 Implemented LoRa radio for long-range, low-power, infrastructure-free communication
- ⚡ Hosted on ESP32 — ultra low-cost, portable, deployable anywhere
